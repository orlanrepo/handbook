# network

Check network interfaces

```text
ifconfig 
ip a
```

Routing

```text
route
ip route
/sbin/route
```

ARP info

```text
arp -a
ip neigh
```

Network information

```text
netstat -ano
netstat -antup
ss -tulw
ss -anp

// details
-a display all sockets
-n don't resolve names
-o display timers
-t tcp
-u udp
-p display PID/Program name for sockets
```
